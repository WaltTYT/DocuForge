# DocuForge 前端产品需求文档

## 1. 产品概述

DocuForge 是一个文档格式互转工具的 Web 前端界面，为用户提供图形化的文档转换体验。
- 主要目的：让非技术用户也能轻松使用文档转换功能
- 解决的问题：替代命令行操作，提供直观的拖拽上传和转换流程
- 目标用户：需要频繁转换文档格式的办公人员、学生、教师

## 2. 核心功能

### 2.1 功能模块

1. **首页**：文件上传区、格式选择、转换按钮、转换历史
2. **转换结果页**：转换进度、下载链接、错误提示

### 2.2 页面详情

| 页面名称 | 模块名称 | 功能描述 |
|---------|---------|---------|
| 首页 | 文件上传区 | 支持拖拽上传和点击选择，显示文件名和大小，支持多文件批量上传 |
| 首页 | 格式选择 | 下拉菜单选择目标格式（.docx/.pdf/.pptx/.md），显示支持的转换路径 |
| 首页 | 转换按钮 | 一键转换，转换中显示进度条，完成后显示下载按钮 |
| 首页 | 转换历史 | 显示最近 10 次转换记录，包含文件名、转换时间、状态，可重新下载 |
| 首页 | 格式说明 | 展示支持的格式和转换路径，帮助用户了解功能 |

## 3. 核心流程

用户使用流程：
1. 用户打开网页，看到文件上传区域
2. 拖拽或点击上传文档文件（支持 .docx/.pdf/.pptx/.md）
3. 选择目标格式（系统自动过滤不支持的转换）
4. 点击"开始转换"按钮
5. 系统调用后端 API 执行转换
6. 转换完成后显示下载按钮
7. 用户点击下载转换后的文件

```mermaid
graph LR
    A[用户打开页面] --> B[上传文件]
    B --> C[选择目标格式]
    C --> D[点击转换]
    D --> E[后端处理]
    E --> F[转换完成]
    F --> G[下载文件]
```

## 4. 用户界面设计

### 4.1 设计风格

- **主色调**：深蓝色 (#1e3a8a) + 活力橙 (#f97316) 作为强调色
- **背景色**：浅灰白 (#f8fafc) 营造清爽感
- **按钮风格**：圆角矩形，主按钮使用渐变蓝色，悬停时有轻微上浮效果
- **字体**：标题使用 "Outfit"（现代几何风格），正文使用 "Noto Sans SC"（中文友好）
- **布局风格**：卡片式布局，大圆角（16px），柔和阴影
- **图标风格**：使用 Lucide React 图标库，线条简洁

### 4.2 页面设计概览

| 页面名称 | 模块名称 | UI 元素 |
|---------|---------|--------|
| 首页 | 文件上传区 | 虚线边框拖拽区，支持点击，显示文件图标和名称，删除按钮 |
| 首页 | 格式选择 | 下拉选择器，显示格式图标（Word/PDF/PPT/MD），禁用不支持的选项 |
| 首页 | 转换按钮 | 大号主按钮，转换中显示加载动画和进度百分比 |
| 首页 | 转换历史 | 列表卡片，每项显示文件图标、文件名、时间、状态标签、下载按钮 |
| 首页 | 格式说明 | 网格卡片展示支持的转换路径，带箭头图标指示方向 |

### 4.3 响应式

- 桌面优先设计，最小宽度 1024px
- 平板适配：768px - 1024px，调整卡片布局为两列
- 移动端：小于 768px，单列布局，简化上传区交互

### 4.4 交互细节

- **文件拖拽**：拖入时边框高亮，显示"释放以上传"提示
- **格式选择**：根据上传文件格式自动过滤可选目标格式
- **转换进度**：进度条从 0% 到 100%，完成后显示对勾图标
- **错误处理**：转换失败时显示红色错误提示卡片，提供重试按钮
- **动画效果**：页面加载时卡片依次淡入，按钮悬停有缩放效果
